# Installation

Velair can be installed through HACS as a custom repository or manually by copying the integration folder.

## HACS Custom Repository

1. Open HACS.
2. Open **Custom repositories** from the menu.
3. Add the Velair repository URL: `https://github.com/cgonfer/velair`.
4. Select **Integration**.
5. Install Velair.
6. Restart Home Assistant.
7. Go to **Settings > Devices & services**.
8. Add the Velair integration.
9. Select the `climate.*` entities Velair may manage.

## Manual Installation

Manual installation is normally only needed when you do not use HACS or when you want to test a GitHub Actions or GitHub Release build.

Release assets include `velair-custom-component-<version>.zip`. Extract it so this folder exists:

```text
<home_assistant_config>/custom_components/velair
```

For a repository checkout, copy:

```text
custom_components/velair
```

to:

```text
<home_assistant_config>/custom_components/velair
```

Restart Home Assistant and add Velair from **Settings > Devices & services**.

## GitHub Actions Artifacts

Every successful CI run uploads temporary artifacts:

- `velair-custom-component`: the complete `custom_components/velair` folder with the frontend bundle generated by that run.
- `velair-lovelace-resource`: the compiled `velair-card.js` files for advanced manual Lovelace resource testing.

Tagged release runs also attach zip files to the GitHub Release. Prefer HACS for normal use; use these artifacts when you want to test a specific workflow run or install manually.

## Optional Lovelace Card

The sidebar panel is registered automatically. The Lovelace card is optional.

Add the integration-served resource:

```yaml
url: /velair_frontend/velair-card.js
type: module
```

Then add one or more Velair cards. Each card uses the same backend data and visual components as the sidebar panel.

```yaml
type: custom:velair-card
view: overview-status
```

Supported `view` values:

- `overview-status`: scheduler state and pause/stop/resume controls.
- `overview-boosts`: active boosts.
- `overview-events`: next events.
- `overview-timeline`: today's timeline.
- `overview-zones`: zone overview.
- `schedules`: full schedule editor.

## Updating

After updating Velair:

1. Restart Home Assistant.
2. Hard refresh the browser or reload the Home Assistant companion app.
3. Confirm the Settings maintenance section shows the expected integration version.
